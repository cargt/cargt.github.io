Introduction
============

Welcome to the cargt project documentation. This page provides an overview of the project.

About This Project
------------------

This is a technical documentation site built using ESP-Docs and deployed to GitHub Pages.

ESP-Docs Features
-----------------

ESP-Docs is a documentation framework that provides:

* **Sphinx-based**: Built on the powerful Sphinx documentation generator
* **Multi-language support**: Easy management of documentation in multiple languages
* **Rich formatting**: Support for code blocks, tables, images, and more
* **API documentation**: Integration with Doxygen for API docs
* **Custom extensions**: ESP-specific extensions for enhanced documentation

Documentation Structure
-----------------------

The documentation is organized as follows:

* **docs/**: Root documentation directory
* **docs/en/**: English documentation
* **docs/en/conf.py**: Language-specific configuration
* **docs/conf_common.py**: Common configuration shared across languages
* **docs/requirements.txt**: Python dependencies
