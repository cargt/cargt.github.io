Introduction
------------

This documentation is powered by ESP-Docs, a documentation framework built on Sphinx and designed for technical documentation.

Features
--------

* Clean and professional documentation
* Support for multiple languages
* Easy to maintain with reStructuredText
* Automated deployment with GitHub Actions
* Hosted on GitHub Pages

Getting Started
---------------

To contribute to this documentation:

1. Clone the repository
2. Install dependencies: ``pip install -r docs/requirements.txt``
3. Build the docs: ``cd docs/en && sphinx-build -b html . ../_build/html``
4. View the output in ``docs/_build/html/``
